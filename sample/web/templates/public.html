Welcome to Maildoor Sample application

This is a sample application to show how to implement Maildoor email authentication for your application.

It has two pages:

Welcome page
The page where you are now, where users will land.
Private Page
A page that should only be accessed by logged in users.

Besides the application specific pages, Maildoor adds authentication related endpoints:

(GET:/auth/login) Login page
The login page.
(POST /auth/send) The email send endpoint
Where the email is validated and sent.
(GET /auth/validate/) Validation endpoint
Where the email token is validated
(DELETE /auth/logout) Logout endpoint
Logout endpoint
